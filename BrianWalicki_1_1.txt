Program Description: 
Sells tickets and keeps track of buyers. Specify maximum tickets in the parameter (tickets) of purchase_sys.

---

Logical Steps:  
1. Call purchase_sys with maximum number of tickets in the parameter.
2. Loops until tickets reaches zero
3. Ask for how many tickets the user would like to purchase.
4. If request is legal. If within 0<=x<=4, not greater than the remaining number of tickets, and it is an integer
5. subtracts from tickets.
5. Prints remaining tickets.
6. Once tickets reaches 0, prints total buyers and ends the program via break.

---

Variables:  
1. [Variable 1: buyers: the number of people who bought tickets
2. [Variable 2: tickets: the total number of tickets remaining

---

Functions:
1. Function Name: purchase_sys  
   Description: facilitates buying and tracking of tickets. Also tracks number of buyers
   Parameters: tickets, the total number of tickets to be purchased
   Returns: None: Prints number of tickets remaining. At the end it prints number of buyers
